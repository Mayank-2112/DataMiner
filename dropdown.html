Logical Functions

AND

The AND function is used to check if all specified conditions are true. It returns TRUE if all conditions are met; otherwise, it returns FALSE.

AND(logical1, logical2, ...)

=AND(A1>10, B1<20)

OR

Returns TRUE if any argument evaluates to TRUE.

=OR(logical1,logical2,...)

=OR(A2>=10,B2<5)

NOT

Returns the reserved local value of its argument . i.e. If the argument is FALSE then TRUE is returned and vice versa.

=NOT(LOGICAL)

=NOT(A2>=10)

Financial Functions

FV

The FV function calculates the future value of an investment or loan, which is the value of an investment at a specified future date.

FV(rate, nper, pmt, [pv], [type])

=FV(0.06, 10, -500, -1000)

NPV

The NPV function calculates the net present value of a series of cash flows, which represents the difference between the present value of cash inflows and outflows.

NPV(rate, value1, [value2], ...)

=NPV(0.1, -1000, 500, 300, 400)

PMT

The PMT function calculates the periodic payment for a loan or investment based on a constant interest rate and equal payments.

PMT(rate, nper, pv, [fv], [type])

=PMT(0.05, 10, 10000)

PV

The PV function calculates the present value of an investment or loan, which is the current worth of a future stream of cash flows.

PV(rate, nper, pmt, [fv], [type])

=PV(0.08, 5, -1000)

RATE

The RATE function calculates the interest rate per period for an investment or loan.

RATE(nper, pmt, pv, [fv], [type], [guess])

=RATE(10, -500, 10000)

Math Functions

AVERAGE

The AVERAGE function calculates the arithmetic mean of a range of numbers.

AVERAGE(number1, number2, ...)

=AVERAGE(A1:A5)

POWER

The POWER function raises a number to a specified power

POWER(number, power)

=POWER(A1, 2)

ROUND

The ROUND function rounds a number to a specified number of decimal places

ROUND(number, num_digits)

=ROUND(A1, 2)

SQRT

The SQRT function calculates the square root of a number

SQRT(number)

=SQRT(A1)

SUM

The SUM function is used to add up a range of numbers

SUM(number1, number2, ...)

=SUM(A1:A5)

Trigonometric Functions

COS

The COS function returns the cosine of an angle in radians.

COS(angle)

=COS(45)

COSH

Returns the hyperbolic cosine of a number.

COSH(number)

=COSH(45)

SIN

The SIN function returns the sine of an angle in radians.

SIN(angle)

=SIN(45)

SINH

Returns the hyperbolic sine of a number.

SINH(number)

=SINH(45)

TAN

The TAN function returns the tangent of an angle in radians.

TAN(angle)

=TAN(70)

TANH

Returns the hyperbolic tangent of a number.

TANH(number)

=TANH(56)

Lookup and Reference Functions

HLOOKUP

The HLOOKUP function works similarly to VLOOKUP but searches for a value in the top row of a table and returns a value from the same column in a specified row

HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])

=HLOOKUP(A2, B2:D10, 3, FALSE)

VLOOKUP

The VLOOKUP function searches for a value in the leftmost column of a table and returns a value in the same row from a specified column

VLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])

=VLOOKUP(A2, B2:D10, 3, FALSE)

INDEX

The INDEX function returns the value of a cell in a specified row and column within a range

INDEX(array, row_num, [column_num])

=INDEX(A1:D10, 3, 2)

INDIRECT

The INDIRECT function returns the value specified by a text string as a reference. It is often used to create dynamic references in formulas.

INDIRECT(ref_text, [a1])

=INDIRECT("A1")

MATCH

The MATCH function searches for a specified value in a range and returns the relative position of that value

MATCH(lookup_value, lookup_array, [match_type])

=MATCH(A2, B2:B10, 0)

Engineering Functions

DEC2BIN

The DEC2BIN function in Excel is used to convert a decimal number into its binary representation. It takes a decimal number as input and returns a string that represents the binary equivalent of that number.

DEC2BIN(number, [places])

DEC2BIN(A1, 8)

DEC2BIN

The DEC2OCT function in Excel is used to convert a decimal number into its octal representation. It takes a decimal number as input and returns a text string that represents the octal equivalent of that number.

DEC2OCT(number)

DEC2OCT(A1)

DELTA

The DELTA function in Microsoft Excel is used to compare two values and returns 1 if they are equal, and 0 if they are not equal. It is primarily used as a logical function to test for equality.

DELTA(number1, number2)

DELTA(A1, B1)

HEX2DEC

The HEX2DEC function in Excel is used to convert a hexadecimal (base-16) number into its decimal (base-10) representation. It takes a hexadecimal number as input and returns the equivalent decimal value.

HEX2DEC(number)/p>

HEX2DEC(A1)

OCT2DEC

The OCT2DEC function in Excel is used to convert an octal number into its decimal representation. It takes an octal number as input and returns the equivalent decimal value.

OCT2DEC(number)

OCT2DEC(A1)
