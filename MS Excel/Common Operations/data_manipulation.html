Data Manipulation

Sorting Data

Here's how you can sort data in Excel:

Select the Data Range: :
Highlight the range of cells or the entire table that you want to sort.

Access the Sort Dialog Box:
Go to the "Data" tab on the Excel ribbon and click on the "Sort" button. Alternatively, you can right-click anywhere within the selected range and choose "Sort" from the context menu.

Choose the Sorting Criteria:
In the Sort dialog box, you will see a list of columns in your data range. Select the column by which you want to sort your data. You can sort by a single column or multiple columns in a specific order.

Set the Sort Order:
For each column, you can specify whether to sort in ascending (smallest to largest) or descending (largest to smallest) order. Use the options provided in the Sort dialog box to select the desired order.

Perform the Sort:
Once you have set your sorting criteria, click the "OK" button to apply the sort. Excel will rearrange the data in the selected range based on the specified sorting order.

Filtering Data

Here's how you can filter data in Excel:

Select the Data Range: :
Highlight the range of cells or the entire table that you want to filter.

Access the Filter Options:
Go to the "Data" tab on the Excel ribbon and click on the "Filter" button. Alternatively, you can right-click anywhere within the selected range and choose "Filter" from the context menu.

Apply Filters:
Excel will add filter arrows to the header row of each column in your data range. Click on the filter arrow for the column you want to filter by.

Set Filter Criteria:
In the drop-down menu that appears, you can choose one or more filter criteria. For example, you can select specific values, sort the values in ascending or descending order, or apply text filters such as "contains," "does not contain," "begins with," or "ends with."

Filtered Data View:
Once you have applied the desired filters, Excel will display only the rows that meet the specified criteria. The rows that don't meet the criteria will be temporarily hidden from view. You can work with the filtered data, perform calculations, copy, or analyze it as needed.

Remove Filters:
To remove the filters and display the entire dataset again, go to the "Data" tab and click on the "Filter" button. Alternatively, right-click anywhere within the filtered range and choose "Filter" from the context menu to toggle off the filter.

Removing Duplicates

Here's how you can remove duplicates in Excel:

Select the Data Range: :
Highlight the range of cells or the entire table that contains the data from which you want to remove duplicates.

Access the Remove Duplicates Dialog Box:
Go to the "Data" tab on the Excel ribbon and click on the "Remove Duplicates" button. Alternatively, you can right-click anywhere within the selected range and choose "Remove Duplicates" from the context menu.

Choose Columns:
In the Remove Duplicates dialog box, Excel will display the column headers from your selected range. By default, all columns are selected. If you only want to check for duplicates in specific columns, uncheck the columns you want to exclude from the duplicate check.

Remove Duplicates:
Once you have chosen the columns and specified the criteria, click the "OK" button to remove duplicates. Excel will scan the selected range and remove any duplicate rows, keeping only the first occurrence of each unique value.

Review Results:
After removing duplicates, Excel will display a message indicating the number of duplicate values found and the number of unique values remaining. The duplicate rows will be removed, and the remaining data will be adjusted accordingly.
